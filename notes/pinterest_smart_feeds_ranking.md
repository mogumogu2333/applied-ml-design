# Building a smarter home feed

这是Pinterest 2014年的文章，介绍home feed排序的逻辑，多个召回源(people, brand user following, interests)的融合和排序。
设计目标：
1. 不同召回源的pin要混合在一起
2. 低质量的、过时的pin要过滤掉，不再显示
3. best-first order而不是newest-first order
